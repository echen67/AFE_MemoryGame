Word Find

Help

In the 'File Upload' page, you can create new word lists by uploading text files. You can switch to 'Make Custom List' mode by clicking the tab at the top right corner.

Before you start:

The 'File Upload' page takes in a text file in a certain format. Be sure your text file follows the following format:

  • Each word is on its own line
  • No special characters or numbers are included
How to Upload File:
  1. Press the 'Browse' button to open your file explorer
  2. Selected the formatted word list file and click 'Open.'
  3. The location of the file and the file name will automatically be filled.
  4. You can rename the word list in the 'Display Name'
  5. Press 'Upload' to create the new list
  • Press the '+' button to add multiple files to upload at once. (NOTE:Each file is a separate word list.)
  • Press 'X' button to remove file upload instance.
  • Recommended to name the puzzle something unique and descriptive.
